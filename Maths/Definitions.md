> [!cite] Indicator Function
>Its a simple membership check. For a set $A$, the indicator function $\mathbb{I}_{A}(x)$ is defined as:
>
>$$\mathbb{I}_A(x) = \begin{cases} 1 & \text{if } x \in A \\ 0 & \text{if } x \notin A \end{cases}
>$$
^def-indicator-function

>[!cite] Kronecker delta function
>Its an equality check. For two values $x, y$, the Kronecker delta function $\delta(x, y)$ is defined as:
>
>$$\delta(x, y) = \begin{cases} 1 & \text{if } x = y \\ 0 & \text{if } x \neq y \end{cases}
>$$
^def-krondelta-function

>[!cite] Heaviside step function
>The Heaviside step function is  defined as: $$h(z) = \begin{cases} 1 & \text{if } z > 0 \\ 0 & \text{if } z \le 0 \end{cases}$$
^def-heaviside-step-function

>[!cite] Homogeneous Function
>In general, a function $f$ is a $k$-degree homogeneous function, if scaling the input by $\alpha$ scales the output by $\alpha^k$:
>$$f(\alpha . x) = \alpha^k f(x)$$
^def-homogeneous-function
